# ChangelogAI 📋✨

**AI-powered changelog generator** — paste a git diff or commit log, get a clean, formatted changelog instantly.

🔗 **Live:** [changelog-ai on GitHub Pages](https://writsop.github.io/changelog-ai/)

## Features

- 🤖 **AI-Powered** — Uses Claude/GPT to parse raw diffs into human-readable changelogs
- 🎨 **Multiple Formats** — Markdown, Keep a Changelog, Conventional Commits
- ⚡ **Instant** — No signup, no API keys needed, runs in your browser
- 🌙 **Dark Mode** — Beautiful dark theme by default
- 📋 **One-Click Copy** — Copy formatted output instantly
- 🆓 **100% Free** — No paywalls, no limits

## How It Works

1. Paste your `git log`, `git diff`, or commit messages
2. Click "Generate Changelog"
3. Get a clean, categorized changelog (Added, Fixed, Changed, etc.)
4. Copy and paste into your CHANGELOG.md

## Use Cases

- **Release notes** for GitHub/GitLab releases
- **Sprint summaries** from messy commit histories
- **Client updates** from technical git logs
- **Open source projects** maintaining CHANGELOG.md

## Tech Stack

- Vanilla HTML/CSS/JS (zero dependencies)
- Client-side AI processing
- GitHub Pages hosting
